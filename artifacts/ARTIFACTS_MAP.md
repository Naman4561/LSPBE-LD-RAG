## Taxonomy

- `artifacts/current/`: current serious-redo artifacts and release summaries.
- `artifacts/support/cache/`: resumable caches and state files, separated from primary result artifacts.
- `artifacts/support/logs/`: run logs, separated from primary result artifacts.
- `artifacts/archive_serious_redo/`: archived serious-redo smoke or low-value byproducts.
- `artifacts/legacy_pre_redo/`: pre-redo historical artifacts.
- `artifacts/archive_misc/`: reserved overflow archive area.
